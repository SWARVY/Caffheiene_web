## 6주차 요약

### 사이드 프로젝트 참가

제가 어느정도 기술스택이 확립되고 나서부터는 진행했던 프로젝트가 블로그 프로젝트밖에 없어서 백엔드와 협업한 사이드 프젝이 있었으면 좋겠다.. 라는 생각이 들었습니다.
기존 팀 프로젝트로 진행했던건 JS-React 기반의 프로젝트였기때문에 하다못해 TS-React, TS-Next의 조그만한 규모의 사이드 프젝을 하나 있었으면 좋겠더라구요

그렇게 생각하다가 마침 구름톤 커뮤니티에서 2기분들이 진행하시는 프로젝트의 인원을 구한다고 하길래 바로 DM을 드려봤고, 이 프로젝트에 참가하게 되었습니다.
이미 MVP까지는 완료되어있는 프로젝트였고 제가 담당할 부분은 로직의 리팩토링 + 추가 기능이 있다면 논의 후 구현 정도입니다.

기술스택은 React, Tailwindcss, ShadcnUI, Zustand 정도인 것 같은데 제가 Shadcn하고 Zustand를 안써봐서 해당 내용은 공부를 좀 해야할 것 같더라구요!
어제 프론트 개발자분과 이야기하며 로직에 대해서 이야기를 나눠봤는데, 지향점이 많이 유사한 것 같아서 프로젝트 하는데에도 별 지장이 없을 것 같다는 생각이 들었습니다.

꾸준히 인강 들으면서 학습중이니, 배운 내용을 블로그 외 프로젝트에 적용해보는 시간을 한번 가져봐야겠습니다 :)

### 팀 네이버 서류 지원 완료

네이버 자소서를 다 써서 제출은 했는데 솔직히 긴가민가하네요..

첫 취준이기도 하고, 솔직히 1트인데 바로 대기업에 갈 수 있을거라고도 생각하지 않습니다. 심지어 **네이버**인데요. 이번 목표는 대기업 코테를 직접 경험해보자에 의의를 두려고 합니다.
뭐 최근들어 코테 공부를 열심히 하긴 해서 만약에.. 진짜 만약에 면접이라도 갈 수 있다면 제게는 둘도 없는 큰 기회가 될 것 같습니다!

당장 이번주 토요일날 코테인데 너무 떨리네요 흐흐..

### 내게 부족한 것들

이번에 존안님의 리액트 강의를 들으면서, 그래도 나 리액트 어느정도 아는데? 라고 생각했던 지난날의 자신감이 많이 와장창 부서져 내리는 것 같습니다.

저는 요즘 대세인 TDD도 잘 할줄 모르고.. (Jest 쓸줄은 아는데 react로 테스트 해본 적은 없습니다.) Github Action도 그냥 굴러다니는 yaml 코드 긁어다가 쓰는게 전부고, aws s3은 뭐 써본적도 없네요
그래도 다행인건 강의로 한번씩 다 찍먹은 해볼 수 있다는거네요. 역시 공부하면 할수록 아는건 '아는게 별로 없다' 라는 사실이 아닌가 싶어요

열심히 강의 들어서 또 이거 기반으로 사이드 프젝을 한번 해보고싶네요. 이번 7주차도 이 마음을 잃지 않고 달려야겠습니다!
